# WhatsApp QR Builder

This folder contains the app that creates a QR code for a WhatsApp chat.

## Run

### Windows

Double-click `start_app.bat`

### macOS / Linux

```bash
chmod +x start.sh
./start.sh
```

## What it does

- Creates a QR code that opens a WhatsApp chat
- Supports an optional pre-filled message
- Shows a QR preview
- Saves the QR as PNG
- Copies the generated link

## Build EXE

On Windows, double-click `build_exe.bat`

The generated file will be:

- `dist\WhatsAppQR.exe`
