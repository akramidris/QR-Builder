# Phone Contact QR Builder

This folder contains the app that creates a QR code for a phone contact card.

## Run

### Windows

Double-click `start_app.bat`

### macOS / Linux

```bash
chmod +x start.sh
./start.sh
```

## What it does

- Creates a standard `vCard` QR code
- Shows a QR preview
- Saves the QR as PNG
- Exports the contact as a `.vcf` file

## Build EXE

On Windows, double-click `build_exe.bat`

The generated file will be:

- `dist\PhoneContactQR.exe`
